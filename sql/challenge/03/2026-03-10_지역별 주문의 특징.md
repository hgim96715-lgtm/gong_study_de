## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
SELECT
	region,
	COUNT(DISTINCT order_id) FILTER (WHERE category = 'Furniture') AS "Furniture",
	COUNT(DISTINCT order_id) FILTER (WHERE category = 'Office Supplies') AS "Office Supplies",
	COUNT(DISTINCT order_id) FILTER (WHERE category = 'Technology') AS "Technology"
FROM records
GROUP BY region
ORDER BY region;
```

##  오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분:**
	- `records` 테이블에는 **하나의 주문(order_id)이 여러 행으로 존재할 수 있어** 동일한 주문이 **중복 집계**되었다.
    
-  **새로 알게 된 함수/꿀팁:**
	- `COUNT(DISTINCT column)`특정 컬럼의 **중복을 제거한 뒤 개수 계산**할 때 사용한다.!!!!!!!! 까먹지마
