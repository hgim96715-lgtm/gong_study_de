## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
SELECT g.name
FROM games g
JOIN platforms p ON g.platform_id = p.platform_id
WHERE g.year >= 2012
AND p.name IN (
'PS3','PS4','PSP','PSV',
'Wii','WiiU','DS','3DS',
'X360','XONE'
)
GROUP BY g.name
HAVING COUNT(DISTINCT 
    CASE
        WHEN p.name IN ('PS3', 'PS4', 'PSP', 'PSV') THEN 'Sony'
        WHEN p.name IN ('Wii', 'WiiU', 'DS', '3DS') THEN 'Nintendo'
        ELSE 'Microsoft'
    END
) >= 2;
```

##  오답 노트 & 배운 점 (Retrospective)

-  **내가 실수한 부분:**
	- 플랫폼 계열을 `CASE`로 분류했지만, **메이저 플랫폼만 필터링하지 않아 다른 플랫폼이 Microsoft로 포함되는 오류**가 발생했다.
	- `HAVING` 절에서도 **`CASE` 표현식** 을 쓸수있다는 생각이 잘 안들어서 CTE를 써야 하나 고민하며 당황했다.
    
-  **새로 알게 된 함수/꿀팁:**
	- `HAVING` 절에서도 **`CASE` 표현식을 직접 사용할 수 있다.**
	- `CASE`에서 `ELSE`를 사용할 경우 **조건에 포함되지 않은 값들이 의도치 않게 같은 그룹으로 묶일 수 있기 때문에**, 필요한 값은 **미리 `WHERE`에서 필터링하는 것이 안전하다.**
