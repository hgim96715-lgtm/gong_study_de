## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
SELECT od.order_purchase_timestamp::DATE AS dt,
       ROUND(SUM(op.payment_value), 2) AS revenue_daily,
       COUNT(DISTINCT od.order_id) AS pu,
       ROUND(SUM(op.payment_value) / COUNT(DISTINCT od.order_id), 2) AS arppu
FROM olist_orders_dataset od
JOIN olist_order_payments_dataset op 
  ON od.order_id = op.order_id
WHERE od.order_purchase_timestamp::DATE >= '2018-01-01'
GROUP BY dt
ORDER BY dt;
```

## 더 나은 풀이가 있다면?

```sql
-- 더 나은 풀이가 있을경우 
WITH order_revenue AS (
    SELECT 
        od.order_id,
        od.order_purchase_timestamp::DATE AS dt,
        SUM(op.payment_value) AS revenue
    FROM olist_orders_dataset od
    JOIN olist_order_payments_dataset op
      ON od.order_id = op.order_id
    WHERE od.order_purchase_timestamp >= '2018-01-01'
    GROUP BY od.order_id, dt
)

SELECT dt,
       COUNT(order_id) AS pu,
       ROUND(SUM(revenue),2) AS revenue_daily,
       ROUND(SUM(revenue)/COUNT(order_id),2) AS arppu
FROM order_revenue
GROUP BY dt
ORDER BY dt;
```
