##  해결 전략 (Code Before Think)

> 데이터베이스를 조회해 게임 배급사로 참여한 게임이 10개 이상인 회사의 이름을 출력하는 쿼리를 작성해주세요.

1. **타겟 데이터:** (어떤 테이블에서 무엇을 뽑아야 하는가?)
   -  `games` 테이블
   - `companies` 테이블
   - `publisher_id`게임 배급사 ID
   - `company_id` : 회사 ID
   - `name` :회사이름
1. **조건 분석:**
   - 회사 이름을 출력 -> companies가 메인
   - 게임 배급사 참여 게임 10개이상 > 이건 id로 COUNT > publisher_id로 GROUP 
   - JOIN , publiser_id, company_id 
1. **사용할 문법:**
   - CTE
   - JOIN
   - HAVING
---
## 정답 쿼리 (Solution)

```sql
-- 여기에 작성한 쿼리를 붙여넣으세요.
WITH pub_cnt AS(
	SELECT COUNT(game_id),publisher_id
	FROM games
	GROUP BY publisher_id
	HAVING COUNT(game_id)>=10
)

SELECT c.name
FROM companies c
JOIN pub_cnt pc ON c.company_id=pc.publisher_id
```
