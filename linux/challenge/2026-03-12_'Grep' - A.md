## ✅ 정답 스크립트 (Solution)

```bash
#!/bin/bash
grep -iE "the|that|then|those"
```

---
## 오답 노트 & 배운 점 (Retrospective)

- **내가 실수한 부분 (Syntax/Spacing):**
	- 처음에는 `{bash}grep -iE "the|that|then|those"` 이렇게 작성했다.
	- 하지만 이 방식은 **단어 전체가 아니라 문자열 일부만 포함되어도 매칭**되기 때문에 문제가 발생했다.
-  **새로 알게 된 명령어/옵션:**
	- `-w` 옵션을 사용하면 **단어 단위로 정확히 일치하는 경우만 검색**할 수 있다는 것을 알게 되었다.

| 옵션   | 의미                         |
| ---- | -------------------------- |
| `-i` | 대소문자 구분 없이 검색              |
| `-E` | 정규식 OR                     |
| `-w` | **단어 단위로 정확히 일치하는 경우만 검색** |
